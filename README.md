# readme

readme is a planned macOS desktop app for turning long articles, essays, letters, and pasted documents into listenable MP3 files using affordable AI text-to-speech voices.

The app is intentionally separate from Speakeasy. Speakeasy is a hold-to-talk dictation tool; readme is a long-form narration and export tool.
